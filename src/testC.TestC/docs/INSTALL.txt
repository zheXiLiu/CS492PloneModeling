testC.TestC Installation
------------------------

To install testC.TestC using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``testC.TestC`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        testC.TestC
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
